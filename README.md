# Api

FastAPI (python) for my personal PostgreSQL database used for my [website](https://combifightet.github.io/)

> https://youtu.be/I8WiIXMDydw?t=2118

## Secrets

_see `./src/config.py`_

| Key | Deafult Value | Description |
| --- | ------------- | ----------- |
| `DATABASE_URL` | `'postgresql+asyncpg://postgres:password@localhost:5432/postgres'` | database url including user and password |
| `HOST` | `'localhost'` | host for the frontend |
| `PORT` | `8000` | port for the frontend |
